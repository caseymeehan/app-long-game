# Long-Game Course Platform

Course platform for "AI For The Long Game" — a 4-week live cohort program.

## Stack

- React Router v7 (SSR) + TypeScript
- Supabase (PostgreSQL + Auth)
- Tailwind CSS 4 + shadcn/ui
- Railway (auto-deploy from GitHub)

## Development

```bash
pnpm install
pnpm dev
```

## Deployment

Push to `main` — Railway auto-deploys from GitHub.

```bash
git push origin main
```

See [docs/PLATFORM.md](docs/PLATFORM.md) for full documentation.
